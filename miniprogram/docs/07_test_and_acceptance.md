# 07 — 测试与验收

## 回归检查清单

每次发版前执行以下清单。标注 [M] 为必检项，[S] 为可选。

### 搜索功能

- [M] 输入关键词 → 建议列表在 100ms 内出现
- [M] 输入无匹配关键词 → 建议列表为空，不报错
- [M] 清空输入框 → 建议列表消失，搜索结果保留
- [M] 完整输入 + 回车/搜索按钮 → 结果列表正确展示
- [M] 搜索结果卡片显示：公式预览（LaTeX 渲染正确）、标题、标签、难度
- [M] 中文关键词能命中（如"不等式"→I001）
- [M] 拼音全拼能命中（如"budengshi"→I001）
- [M] 拼音首字母能命中（如"bds"→I001）
- [M] 英文关键词能命中（如"tan"、"sin"）
- [M] 公式符号能命中（如"f(x)"、"<"）
- [M] 部分输入能前缀匹配命中
- [M] 空查询不触发搜索
- [M] 结果排序稳定：同 query 多次搜索顺序一致
- [S] 搜索结果数正确（与索引中匹配数一致）

### 建议补全

- [M] 输入第一个字 → 立即出现建议
- [M] 建议列表不超过 8 条
- [M] 点击建议项 → 填入搜索框并触发搜索
- [M] 建议项显示匹配字段和得分
- [M] 快速连续输入不出现闪烁或重复请求

### 详情页

- [M] 搜索卡片点击 → 正确跳转到对应详情页
- [M] 核心公式居中渲染清晰（KaTeX 正确编译）
- [M] 所有 section 正确展开（定义、条件、结论、应用、陷阱）
- [M] 含公式的 mixed-content block 正确混排（行内公式 + 文字）
- [M] PDF 按钮在 pdfAvailable=true 时显示
- [M] PDF 下载 → 缓存 → 打开流程正常
- [M] PDF 不可用时按钮不显示（不报错）
- [M] 收藏星标点击 → 状态即时切换（isFavorited toggle）
- [M] 未登录点收藏 → 弹出登录引导
- [M] 返回按钮 → 回到搜索页且保持之前状态

### 收藏功能

- [M] 收藏列表正确展示已收藏条目
- [M] 按模块筛选正确
- [M] 排序切换正确（最近/标题）
- [M] 取消收藏 → 列表即时移除该条目
- [M] 批量管理模式进入/退出正常
- [M] 从收藏列表进入详情 → 详情页正确展示
- [M] 收藏总数字与个人中心显示一致

### 认证

- [M] 游客状态 → "点击登录"按钮可见
- [M] 微信登录流程完整：点击 → wx.login → token 交换 → 用户信息
- [M] 登录后个人中心显示头像 + 昵称
- [M] Token 过期后自动刷新，不打断用户操作
- [M] Token 刷新失败 → 弹出重新登录提示
- [M] 登出后状态恢复为游客

### 离线/弱网

- [M] 飞行模式下搜索可用（本地索引回退）
- [M] 飞行模式下详情可用（本地内容数据）
- [M] 飞行模式下建议可用（本地建议列表）
- [M] 网络恢复后自动切回远程 API
- [M] 远程 API 超时不阻塞 UI（10s 超时后回退）

### 页面流转

- [M] 启动页秒开 → 自动跳转搜索页（无明显白屏）
- [M] Tab 切换正常：搜索 ↔ 个人中心
- [M] 搜索 → 详情 → 返回：搜索状态保持
- [M] 个人中心 → 收藏 → 详情 → 返回：收藏列表状态保持
- [M] 个人中心 → 日志 → 日志详情 → 返回：日志列表状态保持
- [S] 页面栈深度不超过 5 层

### 渲染与兼容性

- [M] KaTeX 公式在 iOS 微信中渲染正确
- [M] KaTeX 公式在 Android 微信中渲染正确
- [M] 不同屏幕尺寸下布局不错乱（iPhone SE ~ Pro Max）
- [M] 暗黑模式下文字可读（如支持）
- [M] 字体加载不出来时公式有降级显示（不白块）

### 异常数据

- [M] URL 参数被篡改（无效 module/id）→ 跳转错误提示页
- [M] 远程 API 返回格式异常 → 不崩溃，回退本地
- [M] 索引文件加载失败 → 明确错误提示，不白屏
- [M] 内容文件加载失败 → 详情页显示错误信息
- [M] 本地存储满 → PDF 缓存失败有提示

## 验收标准

### 搜索命中

| 场景 | 输入 | 期望最小命中 |
|------|------|-------------|
| 中文精确 | "不等式" | I001 |
| 别名 | "连不等式" | I001 |
| 拼音全拼 | "budengshi" | 至少 I001 |
| 拼音首字母 | "bds" | 至少 I001 |
| 公式符号 | "N<f" | I001（如已索引） |
| 前缀 | "不等" | I001 |
| 混合 | "tanx 单调" | 对应结论 |

### 补全速度

- 输入字符到建议出现：< 100ms（本地模式）
- 输入字符到建议出现：< 300ms（远程模式，正常网络）

### 排序正确性

- 精确匹配 > 前缀匹配 > 建议兜底
- 同得分下 rank 低的优先（rank=1 是核心结论）
- 同 rank 下 hotScore 高的优先
- 全部相同时 id 字母序稳定

### 页面跳转

- 启动页停留时间 < 300ms
- 搜索→详情跳转 < 500ms（含内容加载）
- 返回操作无白屏/闪烁

### 异常数据

- 任何异常输入（特殊字符、超长字符串、emoji）不导致崩溃
- 远程 API 不可用时搜索仍返回结果
- 无网络时收藏操作为本地乐观更新
